# Notes App — Full Stack

A full stack notes application built with React, Node.js, Express, and SQLite.

## Structure
- `notes-api/` — Express REST API with JWT auth and SQLite database
- `notes-frontend/` — React frontend with Tailwind CSS and React Router

## How to run

**Backend**
```bash
cd notes-api
npm install
node server.js
```

**Frontend**
```bash
cd notes-frontend
npm install
npm run dev
```
